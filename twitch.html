Twitch Streamers

A Web application to show the Twitch Streamers that are online created as part of Intermediate Front End Development projects on Free Code Camp

I used both jQuery & Angular JS in this project. For CSS, I used SCSS(Bourbon) and tried to implement various features of SASS that I have learned. (variables, mixins, nesting etc).
For getting the Streamers and Channels on Twitch, I used the Twitch API
This project was a good practice for Angular JS concepts and also for reading JSON data from API.

Technologies Used

HTML5, SCSS(Bourbon), jQuery, Angular JS, BootStrap
