Pomodoro Timer

A Web application for creating a Pomodor Clock as part of Advanced Front End Development Projects on Free Code Camp
Simple and effective time management application. Set the timer and get Productive!!

The main challenge that I faced in developing this application was working out the logic to reduce the timer value. It took a lot of trial and error to come up with the function to reduce the timer values.

This application allows you to set timer for 25 mins (Pomodoro), 5 mins (Short Break) & 10 mins (Long Break)

Technologies Used

HTML, CSS, jQuery, BootStrap, Git
